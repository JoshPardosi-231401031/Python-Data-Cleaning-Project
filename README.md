## Project Overview
This project transforms a messy dataset of customer orders into a clean, standardized format ready for analysis. 

## Tools Used
- Python (Pandas, Matplotlib)
- Google Colab

## Key Data Cleaning Steps
1. **Text Standardization:** Fixed text inconsistencies and mapped country categories.
2. **Date Formatting:** Converted mixed date formats into `datetime` and extracted year, month, and day features.
3. **Logical Integrity Checks:** Handled logical errors like zero or negative order quantities.
